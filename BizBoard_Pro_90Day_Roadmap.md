## ⚠️ Common Blocker Contingencies

### Blocker 1: "Not hitting user growth targets"
**Symptoms:** <50% of weekly user target  
**Action Plan:**
1. Analyze drop-off: where are users leaving?
2. Double content output (2x blog posts)
3. Direct outreach: DM 50 potential users
4. Revisit Product Hunt/Reddit for feedback
5. Consider paid promotion earlier (small budget test)

### Blocker 2: "No time to execute everything"
**Symptoms:** Consistently finishing <70% of weekly tasks  
**Action Plan:**
1. Cut P2 tasks entirely
2. Batch similar tasks (e.g., all writing on one day)
3. Use templates for repetitive work
4. Consider hiring VA for $200/month (social scheduling, outreach)
5. Focus on top 1-2 high-impact activities per week

### Blocker 3: "Poor conversion rate (<5%)"
**Symptoms:** Good traffic, low signups  
**Action Plan:**
1. User testing: watch 5 people use landing page
2. Simplify CTA (one clear action)
3. Add social proof above fold
4. Implement exit-intent popup
5. A/B test headlines (3 variations)

### Blocker 4: "Users not converting to paid"
**Symptoms:** Free users plateau, no revenue growth  
**Action Plan:**
1. Survey top 20 active users: why not upgrading?
2. Add usage limits to free tier
3. Create urgency (limited-time discount)
4. Improve value perception (feature comparison)
5. Manual outreach to power users

### Blocker 5: "Paid ads ROI is negative"
**Symptoms:** CAC > $20, LTV < $50  
**Action Plan:**
1. Pause underperforming campaigns immediately
2. Refocus on organic growth channels
3. Test different ad creative (problem-focused vs. solution-focused)
4. Narrow targeting to hyper-specific audiences
5. Increase free trial value to improve conversion
